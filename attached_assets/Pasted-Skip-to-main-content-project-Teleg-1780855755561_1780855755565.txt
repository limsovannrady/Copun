ផែនការអនុវត្ត
Step 1 — Enable Lovable Cloud
បើក backend (Supabase) ដើម្បីយក database, secrets, edge runtime។

Step 2 — Database schema
បង្កើតតារាង (Khmer comments):

bot_settings(key TEXT PK, value JSONB) — KHPAY_API_KEY, CHANNEL_ID, PAYMENT_NAME, MAINTENANCE_MODE, WEBHOOK_SECRET, EXTRA_ADMIN_IDS

bot_account_types(name TEXT PK, price NUMERIC)

bot_accounts(id BIGSERIAL PK, type TEXT FK, code TEXT, created_at) — stock

bot_users(tg_id BIGINT PK, first_name, last_name, username, first_seen)

bot_sessions(tg_id BIGINT PK, state, account_type, quantity, price, total_price, reserved_ids JSONB, transaction_id, md5, qr_sent_at, last_msg_ids JSONB, started_at)

bot_purchases(id BIGSERIAL PK, user_id BIGINT, account_type, quantity, total_price, accounts JSONB, purchased_at)

RLS៖ public anon GET only លើ purchase stats sanitized view; service_role គ្រប់គ្រងគ្រប់យ៉ាងពី server function។

Step 3 — Secrets ត្រូវការ
TELEGRAM_BOT_TOKEN (bot token ពី BotFather)

TELEGRAM_ADMIN_ID (Telegram user ID admin សំខាន់ — default 5002402843)

TELEGRAM_WEBHOOK_SECRET (បង្កើតដោយស្វ័យប្រវត្តិ ឬ user ផ្តល់)

KHPAY_API_KEY (កែបានពី admin dashboard ផងដែរ)

Step 4 — Telegram core library (src/lib/telegram/)
api.ts — tg(method, payload) wrapper ហៅ https://api.telegram.org/bot<TOKEN>/<method>

keyboards.ts — បង្កើត MAIN_KB, ADMIN_KB, ADMIN_SETTINGS_KB, ល។ ដូច bot.js

labels.ts — ប៊ូតុង Khmer labels (កូពី 1-1 ពី bot.js)

qr.ts — qrcode npm → buffer (plain QR; KhPay image URL ប្រើជា priority)

khpay.ts — createKhpayPayment, checkKhpayStatus, khpayRequest

db.ts — repository functions (getSetting/setSetting, accounts CRUD, sessions, purchases)

handlers/ — start.ts, buy.ts, payment.ts, admin/*.ts (មួយឯកសារក្នុងមួយ submenu)

router.ts — handleUpdate(update) បែងចែក message / callback_query

Step 5 — Webhook route
src/routes/api/public/telegram/webhook.ts

POST: ផ្ទៀងផ្ទាត់ header X-Telegram-Bot-Api-Secret-Token ប្រៀបនឹង TELEGRAM_WEBHOOK_SECRET

ហៅ handleUpdate(update)

Return 200 រហ័ស (Telegram timeout 60s)

Step 6 — Background polling for KhPay
Workers មិនអាចមាន setInterval រស់នៅជានិច្ចបានទេ។ ដំណោះស្រាយ៖ check ស្ថានភាពពេលអ្នកប្រើចុច "✅ បាន
